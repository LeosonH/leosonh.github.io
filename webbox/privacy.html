WebBox — Element Editor

Privacy Policy · Last updated April 2026

WebBox currently collects no personal data, sends nothing to any server, and has no analytics or tracking of any kind.

Overview

WebBox is a browser extension that lets you inspect and edit elements on any webpage directly — without opening DevTools. All functionality runs entirely within your browser. Nothing you do with the extension is transmitted, logged, or shared.

Data We Collect

We collect no data. Specifically:

  • No browsing history or page content is recorded
  • No edits you make to page elements are stored or transmitted
  • No personal information is collected at any point
  • No analytics, telemetry, or crash reporting is in place

Local Storage

The only data stored by WebBox is your mode preference (Simple or Advanced), saved locally on your device using chrome.storage.local. This preference:

  • Never leaves your device
  • Is never transmitted to any server
  • Can be cleared at any time by removing the extension

Permissions

WebBox requests the minimum permissions required to function:

  • activeTab Allows the extension to inject the editor into the current tab when you click the toolbar button. Access is granted only to the tab you are actively viewing, only when you invoke the extension.
  • scripting Required to inject the editor script and styles into the active tab. No scripts are loaded from external sources — all code is bundled within the extension package.
  • storage Used exclusively to remember your Simple / Advanced mode preference locally on your device.

Remote Code

WebBox executes no remote code. All JavaScript, CSS, and assets are fully self-contained within the extension package. This complies with Chrome's Manifest V3 requirements and means the extension's behaviour cannot be changed without a published update that you can review.

Third Parties

WebBox has no third-party integrations, SDKs, advertising networks, or data brokers. No data is shared with any third party because no data is collected in the first place.

Changes to This Policy

If this policy is updated in a future version of the extension, the updated date at the top of this page will reflect the change. Continued use of the extension after an update constitutes acceptance of the revised policy.
